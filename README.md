# Cyber News Scraper

A Python-based project that fetches and displays the latest cybersecurity news.

## Features
- Fetch latest cyber news from sources
- Simple web interface using HTML
- Lightweight and easy to run

## Tech Stack
- Python
- Flask
- HTML

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py
